영우가 그때그때 떠올린 아이디어를 짧게 요청하면 `docs/todo/improvements.md`(개선 후보, IMP-NNN)에 항목으로 정리한다.

**Why:** 음성/짧은 텍스트로 던지는 아이디어가 휘발되지 않게 단일 백로그로 모으기 위함.

**How to apply:** "IMP에 추가해줘" 또는 아이디어성 짧은 요청이 오면 improvements.md에 항목 추가(ID 부여, 출처·날짜 기록). 항목 ID로 작업 지시 가능("IMP-001 진행해줘"). [[ssot-docs-sync]]
