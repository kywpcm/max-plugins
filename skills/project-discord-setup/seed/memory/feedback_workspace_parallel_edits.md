영우는 Claude와 별개로 이 워크스페이스의 아키텍처·구조를 직접 정리/재구성한다. "너만 작업하는 게 아니야. 나도 시스템 전체 구조 구성을 위해 작업한다"고 명시함.

**Why:** Claude만 작업하는 게 아니라 영우도 병렬로 작업한다. 삭제/변경을 함부로 되돌리면 영우의 의도를 침범한다.

**How to apply:** `git status`에서 파일 삭제(D)나 예상치 못한 변경을 발견해도 **자동으로 `git checkout` 복구하지 말 것.** 영우의 의도적 정리일 수 있으니, 커밋 대상이 아니면 그대로 두거나 먼저 확인한다. 내가 만들지 않은 파일의 삭제를 임의로 되돌리지 않는다.
